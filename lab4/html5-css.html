HTML5

HTML5 é uma linguagem de marcação usada para estruturar e apresentar conteúdo na World Wide Web.
É a quinta e última versão principal do HTML recomendada pelo World Wide Web Consortium (W3C).

CSS

CSS significa Cascading Style Sheets.
Tem como função descrever como os elementos HTML devem ser exibidos na página web.

Seletor Usado? Explicação
Universal verified_user Aplica o estilo a todos os elementos da página
Tipo verified_user Usa o nome do marcador a estilizar, a regra será aplicada
a todos os elementos que possuam esse marcador
Classe warning Associa um nome a um estilo, o nome é antecedido de "."
Id warning Permite identificar univocamente um elemento do HTML,
o identificador é antecedido de #
Atributo warning Seletores que se aplicam a atributos de elementos,
pode ser apenas referido o atributo, e.g., [target]
Pseudo-classe warning Identificam elementos em determinado estado, na estrutura do documento
Pseudo-elemento warning Seletor para regras que se aplicam a partes de elementos

Combinadores Usado? Explicação
Descendentes warning São separados por espaço:
Ex: div p –todos os p que estão “dentro” de um div
Filhos warning Através do uso de ">":
Ex: div > p –todos os p que estão “logo dentro” de div
Adjacentes warning Através do uso de "~":
Ex: div ~ p –todos os p que estão colocados depois de um div
Imedatamente Adjacentes warning Através do uso de "+":
Ex: div + p –todos os p que estão colocados “imediatamente” depois de um div
Agrupação warning Agrupar os seletores,separando-os por virgulas:
Ex: div,p{ color: red; } -os div e p ficacarão com fonte vermelha
